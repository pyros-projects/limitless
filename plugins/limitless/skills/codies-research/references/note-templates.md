# Note Templates

## Purpose

Use these templates when the session produced something worth preserving locally.

Do not write a note by default for trivial answers.

## 1. Research Memo

Use when:

- the work produced a reusable judgment
- the topic will likely come up again
- future drift is likely

Template:

```md
# {Title}

Snapshot date: YYYY-MM-DD

## Question

{What we were trying to find out}

## Bottom Line

{Short answer}

## What I Found

- {finding}
- {finding}
- {finding}

## Evidence

- {source and why it mattered}
- {source and why it mattered}

## Uncertainty

- {what is still unclear}

## Good Next Directions

- {direction}
- {direction}
- {direction}
```

## 2. Comparison Note

Use when:

- the user compared systems, frameworks, or tools
- the category boundaries were important

Template:

```md
# {Comparison Title}

Snapshot date: YYYY-MM-DD

## Category Split

{Why these things are similar or different in kind}

## Recommendation

{High-level judgment}

## Comparison

| Option | What it is | Strength | Weakness | Best fit |
|--------|------------|----------|----------|----------|
| A |  |  |  |  |
| B |  |  |  |  |

## Drift Risks

- {likely misunderstanding or category trap}

## Good Next Directions

- {direction}
- {direction}
```

## 3. Update Note

Use when:

- the user asked to refresh an existing note
- dates and changed assumptions matter

Template:

```md
# {Title}

Original note: {path or name}
Update date: YYYY-MM-DD

## What Changed

- {change}
- {change}

## What Still Holds

- {still true}

## What Became Stale

- {stale assumption}

## Revised Bottom Line

{updated answer}

## Good Next Directions

- {direction}
- {direction}
```
